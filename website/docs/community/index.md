# Community

Dilder is built in the open, and the community is part of that.

---

## Discord

The primary place for real-time discussion, build help, and sharing your own Dilder.

[Join the Discord Server :fontawesome-brands-discord:](discord.md){ .md-button .md-button--primary }

**Channels include:**

- `#hardware` — wiring questions, component sourcing, soldering tips
- `#software` — code questions, Pi setup, firmware debugging
- `#3d-printing` — case design, print settings, filament recommendations
- `#show-your-build` — post photos and videos of your Dilder
- `#suggestions` — feature requests and ideas
- `#bugs` — report issues in the docs or firmware
