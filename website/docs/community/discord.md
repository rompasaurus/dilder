## Server Structure

### Welcome

| Channel | Purpose |
|---------|---------|
| `#welcome` | Rules, how to get started, role assignment |
| `#introductions` | Say hello and tell us what you're building |

### General

| Channel | Purpose |
|---------|---------|
| `#general` | Main chat — anything project related |
| `#off-topic` | Non-project discussion |

### Build Series

| Channel | Purpose |
|---------|---------|
| `#hardware` | Component questions, wiring help, sourcing tips |
| `#software` | Pi setup, firmware, Python questions |
| `#3d-printing` | Case design, print settings, filament |
| `#show-your-build` | Photos, videos, and demos of your Dilder |

### Development

| Channel | Purpose |
|---------|---------|
| `#dev-log` | Automated updates from GitHub pushes |
| `#suggestions` | Feature requests and ideas |
| `#bugs` | Bug reports for docs and firmware |

### Support

| Channel | Purpose |
|---------|---------|
| `#patreon-chat` | Patreon supporters only — early access discussion |

## Rules (Short Version)

1. Be kind. This is a learning community — no question is too basic.
2. Stay on topic in build channels.
3. No spam, no self-promotion without context.
4. Credit sources when sharing others' work.

Full rules are pinned in `#welcome`.
