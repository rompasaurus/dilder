# TP4056 — LiPo Battery Charger IC

## Quick Reference

| Attribute | Value |
|-----------|-------|
| **Manufacturer** | TPOWER |
| **Part Number** | TP4056 |
| **Function** | Single-cell Li-Ion/LiPo linear charger (CC/CV) |
| **Package** | ESOP-8 (SOP-8 exposed pad) |
| **LCSC** | [C382139](https://www.lcsc.com/product-detail/C382139.html) |
| **Price (qty 5)** | ~$0.07 |
| **Dilder ref** | U2 |

## Key Specifications

| Parameter | Value |
|-----------|-------|
| Charge voltage | 4.2V +/-1% |
| Max charge current | 1A (set by RPROG resistor) |
| Input voltage | 4.0V to 8.0V |
| Charge method | Constant-Current / Constant-Voltage (CC/CV) |
| Trickle charge | 50mA (when VBAT < 2.9V) |
| Charge termination | Current drops to 1/10 of programmed charge current |
| CHRG pin | Open-drain output, LOW = charging |
| STDBY pin | Open-drain output, LOW = charge complete |
| Thermal regulation | Yes (reduces current if junction temp > 120C) |
| Operating temp | -40C to +85C |

## Charge Current Setting

The charge current is set by a single resistor (RPROG) from the PROG pin to GND:

| RPROG Value | Charge Current |
|-------------|----------------|
| 1.2k | 1000 mA |
| 2.0k | 580 mA |
| 3.0k | 400 mA |
| 5.0k | 250 mA |
| 10.0k | 130 mA |

**Dilder uses 1.2k (R1)** for 1A charging of the 1000mAh battery (~1 hour charge time).

## Pin Connections (Dilder Board)

| TP4056 Pin | Pin # | Connect To | Notes |
|------------|-------|------------|-------|
| VCC | 3 | USB 5V (via Schottky) | Charge input from USB-C |
| GND | 2 | GND | Ground |
| BAT | 5 | Battery + (via DW01A/FS8205A) | Battery connection |
| PROG | 7 | 1.2k resistor to GND | Sets 1A charge current |
| CHRG | 1 | Red LED (via 1k resistor) | LOW when charging |
| STDBY | 6 | Green LED (via 1k resistor) | LOW when charge complete |
| CE | 8 | VCC (or control GPIO) | HIGH = charger enabled |
| TEMP | 4 | N/C or NTC thermistor | Battery temperature monitoring (optional) |
