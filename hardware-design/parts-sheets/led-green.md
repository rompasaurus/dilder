# Green LED (0603) — Charge Complete Indicator

| Attribute | Value |
|-----------|-------|
| **Manufacturer** | Everlight |
| **Part Number** | 19-217/GHC-YR1S2/3T |

| **Package** | 0603 (1608 metric) |
| **LCSC** | [C72043](https://www.lcsc.com/product-detail/C72043.html) |

## Key Specifications

| Parameter | Value |
|-----------|-------|
| Color | Emerald Green (518nm) |
| Forward voltage | 3.0V - 3.4V typical |
| Forward current | 20mA max |
| Luminous intensity | 25-63 mcd |

## Circuit

Connected to TP4056 STDBY pin via 1k current-limiting resistor (R3). LED turns ON when battery is fully charged.
