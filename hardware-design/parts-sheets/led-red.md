# Red LED (0805) — Charging Indicator

## Quick Reference

| Attribute | Value |
|-----------|-------|
| **Manufacturer** | Foshan NationStar |
| **Part Number** | NCD0805R1 |
| **Function** | Red LED — indicates battery is charging |
| **Package** | 0805 (2012 metric) |
| **LCSC** | [C84256](https://www.lcsc.com/product-detail/C84256.html) |
| **Price (qty 5)** | ~$0.01 |
| **Dilder ref** | D2 |

## Key Specifications

| Parameter | Value |
|-----------|-------|
| Color | Red (625nm) |
| Forward voltage | 1.8V - 2.2V typical |
| Forward current | 20mA max |
| Luminous intensity | 60-150 mcd |

## Circuit

Connected to TP4056 CHRG pin via 1k current-limiting resistor (R2). LED turns ON when battery is charging.
