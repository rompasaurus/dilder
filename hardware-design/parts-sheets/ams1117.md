## Key Specifications

| Parameter | Value |
|-----------|-------|
| Output voltage | 3.3V fixed |
| Output current | 1A max |
| Dropout voltage | 1.1V @ 1A (Vin must be >= 4.4V for 1A) |
| Input voltage | 4.5V to 15V (recommended) |
| Quiescent current | 5 mA typical |
| Line regulation | 0.2% max |
| Load regulation | 0.4% max |
| Thermal shutdown | Yes (165C) |

## Application Notes

- Input cap: 10uF (tantalum or ceramic) close to VIN
- Output cap: 10uF (tantalum or ceramic) close to VOUT — required for stability
- The AMS1117 regulates the TP4056's output (3.7-4.2V from battery) down to a stable 3.3V for the ESP32-S3 and all digital ICs
- At typical Dilder load (~100mA), dropout is only ~0.3V — battery can run down to ~3.6V before regulation is lost

## Pin Connections

| Pin | Function | Connect To |
|-----|----------|------------|
| 1 (GND/Adj) | Ground | GND |
| 2 (VOUT) | 3.3V output | ESP32-S3 VDD, LIS2DH12 VDD, pull-ups |
| 3 (VIN) | Input voltage | VSYS / TP4056 OUT+ |
| Tab | Ground (exposed pad) | GND (thermal relief to ground plane) |
